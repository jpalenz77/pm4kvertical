# PM4K Vertical UI

Modern vertical navigation interface for PlexMod (PM4K).

## Description

PM4K Vertical UI adds a Netflix-style vertical navigation interface to PlexMod. It's a complementary addon that enhances PM4K with modern vertical scrolling navigation.

## Features

- ✅ **Vertical Navigation**: Modern Netflix-style vertical scrolling
- ✅ **Integrates with PM4K**: Uses PM4K's authentication and libraries
- ✅ **Lightweight**: Small addon that extends PM4K functionality
- ✅ **Modern UI**: Clean interface with smooth animations
- ✅ **Easy to Use**: Just install and launch

## Requirements

- **Kodi 19+** (Matrix, Nexus, or Omega)
- **PlexMod (PM4K)** installed and configured
- PlexMod must be authenticated with your Plex server

## Installation

1. Make sure PM4K is installed from https://pm4k.eu
2. Configure PM4K and sign in to your Plex account
3. Install `script.pm4k.verticalui-1.0.0.zip` from zip file
4. Launch "PM4K Vertical UI" from Programs

## How It Works

This addon:
1. Checks if PM4K is installed
2. Uses PM4K's libraries and authentication
3. Displays your Plex libraries in vertical layout
4. When you select content, opens it with PM4K for playback

## Interface

- **Left Sidebar**: Vertical list of your Plex libraries
- **Main Area**: Grid of content from selected library
- **Smooth Navigation**: Arrow keys to navigate, Enter to select

## Advantages

- No need to configure Plex again (uses PM4K)
- Lightweight - only adds the vertical UI
- Works alongside PM4K's normal interface
- Easy to maintain and update

## Troubleshooting

### "PM4K No Instalado"
Install PlexMod from https://pm4k.eu

### "PM4K No Configurado"
Open PM4K first and sign in to your Plex account

### Content doesn't play
The addon uses PM4K to play content. Make sure PM4K works normally first.

## Development

This is an extension addon that requires PM4K. It doesn't duplicate PM4K's functionality, instead it provides an alternate vertical navigation interface.
